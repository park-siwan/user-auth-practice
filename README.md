### graphQL install

```
npm i @nestjs/graphql @nestjs/apollo @apollo/server graphql
```

### typeORM install

```
npm i --save @nestjs/typeorm typeorm
```

### database install

```
npm i --save (mysql2 or sqlite3 or p)
```

### validator install

```
npm i class-validator joi
```

### auth install

```
npm i bcrypt jsonwebtoken
```
